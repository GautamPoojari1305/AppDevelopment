BookWorm is a modern mobile application built using React Native + Expo that allows users to search, explore, and view detailed information about books. It features Firebase authentication, beautiful UI styling, category browsing, and dark mode support.

 Features
 User Authentication

Sign Up using Email & Password
Login using Firebase Authentication
Logout functionality
User profile page showing:
Name
Email
Dark/Light mode toggle

 Book Search

Search for any book using Google Books API
Automatic results displayed as cards
Clean grid layout (2 columns)

 Book Categories

Scrollable horizontal category list
Each category has an icon
Tapping a category fetches related books
Neon blue themed chips
Smooth layout with no clipping issues

Categories include:
Manga, Romance, Adventure, Fiction, Fantasy,
Science Fiction, Mystery, Thriller, Horror,
Biography, History, Self Help, Poetry

 Book Details Page

Each book card opens a Book Details Screen showing:
Cover image
Title
Authors
Published date
Page count

UI & UX Enhancements

Neon blue glowing buttons
Animated category icons
Smooth scroll layout
Search bar with outlined styling
Cards with rounded corners & shadows
Centered login & signup pages

Clean light & dark themes
Manual Theme Toggle
Switch between Dark and Light mode
Works across all screens
Controlled globally using Context API
