Experiment No:1

Simple Array CRUD Operations 


 How to Run the Program

1.  Clone the repository.
2.  Navigate to the project directory.
3.  Run the script using Node.js.


Program Functionality

The program performs the following operations on an array of objects:

* **Create:** Adds a new item to the array.
* **Read:** Displays all items in the array.
* **Update:** Modifies an existing item in the array.
* **Delete:** Removes an item from the array.

The output picture (`output.png`) in this folder shows the console output of the program, demonstrating each of these operations in action.
