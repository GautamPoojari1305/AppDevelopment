React Native To-Do List App


The app includes the following core functionalities:

* **Add Tasks:** Easily add new tasks to your list using the input field and button.
* **Mark as Complete:** Tap a task to toggle its completion status. Completed tasks are visually marked with a strikethrough.
* **Delete Tasks:** Remove unwanted tasks from the list by long-pressing on them.

---

 How to Run the App

To get this app up and running on your device, follow these steps:

1.  **Install Node.js and Expo CLI:** If you haven't already, ensure you have Node.js and the Expo CLI installed on your computer.
2.  **Clone the Repository:** Download the project files.
3.  **Navigate to the Project Directory:** Open your terminal and change to the project folder.
4.  **Install Dependencies:** Run `npm install` to download all necessary packages.
5.  **Start the Server:** Launch the development server with `expo start`.
6.  **Open the App:** Use the Expo Go app on your phone to scan the QR code displayed in your terminal or web browser. The app will build and run on your device.
